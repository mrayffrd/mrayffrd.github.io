EMU JOKI - Jasa Joki Game

1. Pemesanan & Pembayaran

  • Order via form Website
  • Harga di form = estimasi. Bukan harga final yang dapat dinegosiasi dengan pertimbangan
  • Pelunasan 100% sebelum dikerjakan
  • Pelunasan setelah selesai + ada bukti Screenshot

2. Proses Joki

  • Estimasi Keterlambatan 1-4 hari kerja per 1 pengerjaan joki (tidak termasuk libur)
  • Jika ingin bermain genshin saat proses penjokian harap konfirmasi admin agar tidak bertabrakan
  • Akan dikasih update status dari Website. Jika ingin Screenshot berkala saat penjokian harap konfirmasi

3. Larangan

PENTING:
  • Web EMU tidak pernah meminta Akun atau Password akun
  • Web EMU tidak pernah meminta nomor kartu Kredit atau Debit
  • Web EMU tidak pernah meminta kalian menginstal Software

    • 4. Garansi

    • Garansi Uang Kembali 50% jika melebihi estimasi yang dijanjikan (garansi 1 hari setelah joki selesai) jika tidak claim garansi sampai waktu habis maka garansi hangus
    • Garansi Uang Kembali 50% untuk joki rawat akun jika EMU melakukan absen (garansi 1 hari setelah joki selesai) jika tidak claim garansi sampai waktu habis maka garansi hangus
    • Tidak dapat membatalkan Pesanan yang telah melakukan Proses Pembayaran
